Documentación del Programa

Descripción

Este programa es un editor de tablas en Tkinter que permite manipular datos y exportarlos a formato CSV. Está diseñado para gestionar autómatas finitos y facilitar la visualización de sus estados y transiciones.

Requisitos

  • Python 3.x
  • tkinter
  • Otras dependencias listadas en requirements.txt

Instalación

Para instalar el programa, sigue estos pasos:

  1. Clona el repositorio desde GitHub.
  2. Crea y activa un entorno virtual.
  3. Instala las dependencias utilizando pip install -r requirements.txt.

Uso

Para ejecutar el programa, simplemente corre el archivo principal. Puedes ingresar cadenas y manipular la tabla según sea necesario.

Funciones Principales

Agregar Fila

Permite agregar una nueva fila a la tabla.

Agregar Columna

Permite agregar una nueva columna a la tabla.

Guardar como CSV

Exporta los datos de la tabla a un archivo CSV.
